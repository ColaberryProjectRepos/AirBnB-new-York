Create an association between the Table   


1. Click on the model tab to begin creating an association

 2. Drag the DateKey (Last review) column in the Airbnb NY Table to join the DateKey column in the DimDateTable table to create an association between them.

 3. We renamed the Last Review Column as our Date Column because the table has this column only as a time Dimension and by referring to this column, we created DimDatetable to manipulate the data
